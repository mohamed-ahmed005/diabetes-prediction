# Diabetes Prediction Using Machine Learning

## Overview

This project develops and evaluates machine learning models to predict the likelihood of diabetes based on patient health information. The project follows a complete data science workflow, including data preprocessing, exploratory data analysis (EDA), feature investigation, model training, and performance evaluation.

The goal is to identify individuals at risk of diabetes and demonstrate how machine learning can support healthcare decision-making.

## Dataset

The project uses the Pima Indians Diabetes Dataset, which contains 768 patient records and multiple health-related features such as:

* Glucose Level
* Blood Pressure
* BMI
* Insulin
* Age
* Diabetes Pedigree Function

The target variable indicates whether a patient has diabetes.

## Project Workflow

### Data Exploration and Cleaning

* Examined dataset structure and feature distributions.
* Identified missing or invalid values.
* Performed data preprocessing and preparation for modeling.

### Exploratory Data Analysis (EDA)

* Visualized feature distributions.
* Investigated relationships between variables.
* Analyzed factors associated with diabetes outcomes.

### Machine Learning Models

* Logistic Regression
* Random Forest Classifier

### Model Evaluation

Models were assessed using standard classification metrics and performance evaluation techniques to determine the most effective solution.

## Results

* Logistic Regression achieved approximately 76% accuracy.
* Random Forest provided improved predictive performance and was selected as the preferred model.
* ROC-AUC analysis demonstrated strong classification capability.
